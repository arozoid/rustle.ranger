# rustle.ranger
simplistic yet modern ranger configuration

(ranger is a vim inspired file manager! check it out [https://github.com/ranger/ranger/](here!))

## Installation
(make sure you already have ranger on your system before you continue)

if you're a git user, and you want to keep up with any updates (if there will be any in the future):
```bash
# delete ~/.config/ranger/ (if it already exists)
rm -rf ~/.config/ranger
# clone the repository
git clone https://github.com/arozoid/rustle.ranger ~/.config/ranger/
# start ranger and have fun!
ranger
```

if you're not:

1. click '<> Code' on this repo's homepage (you're probably already here)
2. on the dropdown, click 'Download ZIP' (this downloads the repository without the git config)
3. extract the ZIP file's contents into ~/.config/ranger/
4. start ranger again and enjoy the new look!
